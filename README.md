# miniguia-estudos-notebooklm
Analise das prestações de contas de uma OSC (Organização de Sociedade Civil) com a finalidade de subsidiar a avaliação anual feita pela Comissão de Monitoramento e Avaliação responsável por acompanhar a execução da parceria da OSC com a Prefeirtura. O objetivo é automatizar e facilitar o processo para analise das contas
